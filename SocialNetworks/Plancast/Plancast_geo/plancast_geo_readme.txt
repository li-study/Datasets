
Readme for Plancast dataset network data.

This dataset contain geographic location of social events in plancast.com

1. plancast_event_lon_lat.csv contains the geographic locations for plancast
  social events (if specified). The csv file contains three columns, which
  represent event_id, longitude, and latitude, respectively. That is a line
  "124,-122.426163,37.761471" indicates that event "124"'s venue is located at
  "-122.426163,37.761471".
