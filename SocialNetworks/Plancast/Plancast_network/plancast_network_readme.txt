
Readme for Plancast dataset network data.

This dataset contain the online and offline social network information for data
collected from plancast.com

1. plancast_user_subscription.csv contains the online ``directed'' social
  network among plancast users. The users in the first column subscribe to
  corresponding users in the second column. E.g. a line "12,14" means user "12"
  ``follows'' user "14".

2. plancast_user_event.csv contains all the plancast social event participation
  information. The first column stores user ids and the second column stores
  participated event ids.
